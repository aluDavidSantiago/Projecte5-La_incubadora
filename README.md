<h2>1. Introducción</h2>

<p>
Este proyecto forma parte del ciclo formativo de Sistemas Microinformáticos y Redes (SMX2),
y representa una simulación real del proceso de creación de una empresa tecnológica.
</p>

<p>
A diferencia de proyectos anteriores, el enfoque integra negocio, infraestructura,
organización y defensa profesional de una solución IT.
</p>

<hr>

<h2>2. Contexto del proyecto</h2>

<p>
El proyecto se desarrolla dentro de una incubadora de empresas, donde el alumno adopta el rol
de fundador de una startup tecnológica orientada a clientes B2B.
</p>

<p>
El objetivo es diseñar una empresa realista que resuelva un problema existente,
con una solución técnica viable y defendible.
</p>

<hr>

<h2>3. Objetivos</h2>

<ul>
<li>Diseñar una empresa IT desde cero</li>
<li>Detectar un problema real</li>
<li>Crear una solución técnica integrada</li>
<li>Desarrollar infraestructura IT</li>
<li>Diseñar presencia digital</li>
<li>Definir modelo de negocio</li>
<li>Defender el proyecto</li>
</ul>
